# H46 Sorter 复现步骤分解

## 项目概述

日向坂46（Hinatazaka46）成员排序工具 - 基于 Vue 3 + Vite 开发

## 技术栈

- **前端框架**: Vue 3 Composition API
- **构建工具**: Vite
- **路由**: Vue Router
- **国际化**: Vue I18n
- **图片导出**: html2canvas
- **样式**: 原生 CSS

npm run build
